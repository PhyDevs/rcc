## **Installation**
1.  Clone project
```bash
git clone https://github.com/PhyDevs/rcc.git
```

2.  cd into folder and download dependencies and starting dev server
```bash
cd rcc
yarn install
```

3.  Starting dev server
```bash
yarn start
```
